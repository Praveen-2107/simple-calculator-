# Simple Interest Calculator

This is a simple web application that calculates simple interest. It is very user-friendly.

## Features

*   Calculates simple interest based on principal, rate, and time.
*   Displays the total amount (principal + interest).
*   User-friendly interface.

## How to Use

1.  Open the `index.html` file in a web browser.
2.  Enter the principal amount, annual interest rate, and time in years.
3.  Click the "Calculate" button.
4.  The simple interest and total amount will be displayed.
